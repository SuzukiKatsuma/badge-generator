## Development

### 0. Prerequisites
- Node.js (v22.x or later)
- Yarn

### 1. Installation
```bash
git clone https://github.com/SuzukiKatsuma/badge-generator.git
cd badge-generator
yarn install
```

### 2. Development Server
```bash
yarn dev
```

### 3. Build
```bash
yarn build
```
